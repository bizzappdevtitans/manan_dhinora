This module will make managing the fabrication buisness easier using sale, purchase, manufacturing, automated reminderemail, project and task

Added 3 new fields to Sale, To better explain the project specifination to the end users.

After clicking on confirm Sale will create a project, tasks based on order lines and manufacturing order.

The created project will have the same Name as the customer and the date on which its sale order was created.

And the task will have the name of product to be manufactured along with the customer name, Also added refrence to sale in there.

As for manufacturing order, it will create a purchase order based on the required materials basted on mto routes.

On the purchase order there is a button to the sale order, and a filed called intenal refrence, which will be updated if the scheduled replenishment did replinesh stock.

After manufacturing is over an invoice is created with the specification, as to weather the customer opted in for maintanency service or not.

Based on what the customer asked for, there is a maintance service email, which will be sent every quater.
